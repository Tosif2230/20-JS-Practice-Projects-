# 💱 Currency Converter

A simple and responsive Currency Converter built using **HTML**, **CSS**, and **JavaScript**. This application allows users to convert one currency to another using real-time exchange rates fetched from a Currency Exchange API.

## 🚀 Features

- 🌍 Convert between multiple world currencies
- 📈 Real-time exchange rates using a Currency API
- 🔄 Swap currencies easily
- 📱 Responsive and user-friendly interface
- ⚡ Fast and lightweight

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Currency Exchange API

## 📂 Project Structure

```
03_currency_converter/
│── index.html
│── style.css
│── script.js
└── README.md
```

## ⚙️ How It Works

1. Enter the amount to convert.
2. Select the source currency.
3. Select the target currency.
4. The application fetches the latest exchange rate from the Currency API and displays the converted amount.

## 🔑 API Used

This project uses a Currency Exchange API to fetch live exchange rates.

Replace the API key in `script.js` with your own key if required.

Example:

```javascript
const API_KEY = "YOUR_API_KEY";

Get your api key from exchangerate-api
```
